Please use the following instructions for running the code:
1.	Run initial_trasformation_code.m – This script conducts several transformations and splits the UCI Car Evaluation Dataset into 80% training set and 20% test set.
2.	Run nb_model_selection.m – This script contains the code needed for computing the Naïve Bayes algorithm on the UCI Car evaluation dataset. 
3.	Run random_forest_model_selection.m – This script containts the code needed for computing the Random Forest algorithm on the UCI Car evaluation dataset.
